## Tecnologias:
- Maven
- Java 17
- JavaFX
- JPA/Hibernate
- JUnit
- MySQL

## Como rodar:
1. Certifique-se de possuir o Java, Maven e o MySQL instalados na máquina.
2. Abra o terminal.
3. Digite <code>mvn clean javafx:run</code>
